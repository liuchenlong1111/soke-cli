## ✨ 特色功能

### 1. 多维度筛选
```bash
# 组合多个筛选条件
soke-cli course +list-courses \
  --start-time 1704038400000 \
  --end-time 1735660799000 \
  --status 1 \
  --is-in 1 \
  --category-id "cat001"
```

### 2. 批量查询
```bash
# 一次查询多个用户的学习记录
soke-cli course +list-course-users \
  --course-id "course123" \
  --userid-list "user1,user2,user3"
```

### 3. 时间范围查询
```bash
# 查询特定时间段完成的学习记录
soke-cli course +list-course-users \
  --course-id "course123" \
  --finish-start-time 1704038400000 \
  --finish-end-time 1735660799000
```

### 4. 人脸识别监控
```bash
# 确保学员本人学习
soke-cli course +list-lesson-faces \
  --lesson-id "lesson123"
```

## ⚠️ 重要提示

1. **时间格式**: 必须使用毫秒级Unix时间戳（不是秒）
2. **时间范围**: 课程列表查询的时间范围不能超过365天
3. **分页限制**: 每页最大100条记录
4. **批量限制**: userid-list 最多支持100个用户ID
5. **权限要求**: 所有操作需要 `course:*:readonly` 权限
